# V0.1 Contract

The first implementation target is a univariate Gaussian animal model.

## R Target

```r
fit <- hsquared(
  y ~ sex + age + animal(1 | id, pedigree = ped),
  data = dat,
  family = gaussian(),
  REML = TRUE
)
```

## Julia Target

```julia
fit = fit_animal_model(y, X, Z, Ainv; method = :REML)
```

## R Responsibilities

- Parse the model formula.
- Validate data and pedigree columns.
- Build or request design matrices.
- Marshal a normalized model specification to `HSquared.jl`.
- Return an S3 `hsquared_fit` object.
- Provide R-style summaries and extractors.

## Julia Responsibilities

- Encode IDs.
- Validate and sort pedigrees.
- Build sparse Ainv directly.
- Fit the Gaussian animal model by ML or REML.
- Return a compact result object and diagnostics.

## V0.1 Outputs

- Variance components.
- Heritability.
- Fixed effects.
- EBVs/BLUPs.
- Log-likelihood.
- Convergence diagnostics.
- Minimal fitted metadata.

## Explicit Non-Scope

The following are planned but not v0.1:

- multivariate models;
- `cov = us()`, `cov = fa()`, and `lowrank()`;
- genomic and single-step models;
- GLLVM-style non-Gaussian high-dimensional models;
- GPU paths;
- unusual inheritance systems.
