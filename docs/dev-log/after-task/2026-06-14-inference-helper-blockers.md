## Public claim audit

Rose verdict: clean.

This is a claim-boundary and user-error slice only. It does not add validated
standard errors, confidence intervals, profile likelihoods, likelihood-ratio
tests, model-comparison guidance, or any new estimation capability.

## Tests of the tests

The new focused test constructs a minimal `hsquared_fit` object and calls the
standard R generics directly (`stats::confint()`, `stats::vcov()`,
`stats::profile()`, and `stats::anova()`). The expectations check for the
planned/not-implemented wording, so future accidental partial implementations
or generic fallbacks should fail loudly.
