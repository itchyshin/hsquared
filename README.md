# hsquared

`hsquared` is an R package scaffold for tools to estimate, report, and work
with h-squared quantities.

The package is in early development. Public functions and examples will be
added as the API is designed.

## Installation

```r
# install.packages("pak")
pak::pak("itchyshin/hsquared")
```

## Development

Run the local checks with:

```r
devtools::check()
```
