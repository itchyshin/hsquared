# Quantitative-Genetic Scout Map

Use this as a starting map, not as evidence by itself.

| Reference | Why it matters for hsquared |
| --- | --- |
| ASReml | Proprietary benchmark for REML animal and breeding models. |
| MCMCglmm | Flexible Bayesian animal and evolutionary models. |
| sommer | Open R mixed models with known covariance structures. |
| JWAS.jl | Julia Bayesian genomic prediction and GWAS. |
| BLUPF90 | Production animal-breeding mixed-model software. |
| DMU | Multivariate animal-breeding mixed-model software. |
| WOMBAT | Classical REML covariance-component software. |
| AGHmatrix | A, G, and H relationship construction, including polyploid support. |
| nadiv | Additive and non-additive inverse relationship matrices. |
| XSim.jl | Simulation partner for pedigrees, genomes, and selection response. |
| drmTMB / DRM.jl | Distributional-regression formula and R-Julia twin discipline. |
| gllvmTMB / GLLVM.jl | High-dimensional latent-variable and long/wide documentation discipline. |
